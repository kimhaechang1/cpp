# 📚 개념 정리: C++ STL queue (`std::queue`)

## 1. std::queue 란?
C++ 표준 라이브러리(STL)에서 제공하는 큐(Queue) 컨테이너입니다. `#include <queue>` 헤더를 포함하여 사용할 수 있으며, 데이터가 들어간 순서대로 나오는 FIFO(First In First Out) 구조를 지원합니다.

## 2. 주요 API (메서드) 목록
STL 큐는 멤버 함수를 통해 간단하게 조작할 수 있습니다. 각 함수의 시간 복잡도는 모두 `O(1)`입니다.

- **`push(x)`**: 큐의 맨 뒤(Rear/Tail)에 원소 `x`를 삽입합니다.
- **`pop()`**: 큐의 맨 앞(Front/Head)에 위치한 원소를 제거합니다.
- **`front()`**: 큐의 맨 앞(Front/Head)에 있는 원소의 값(참조)을 반환합니다.
- **`back()`**: 큐의 맨 뒤(Rear/Tail)에 있는 원소의 값(참조)을 반환합니다.
- **`empty()`**: 현재 큐가 비어있는지 여부를 확인합니다. 비어있으면 `true`, 원소가 하나라도 있으면 `false`를 반환합니다.
- **`size()`**: 현재 큐에 들어있는 원소의 총 개수를 반환합니다.

> ⚠️ **핵심 주의사항**: C++ STL의 `pop()` 함수는 원소를 제거하기만 할 뿐, 제거되는 원소의 값을 반환하지는 않습니다. (Java, Python 등과 다른 점입니다!) 따라서 가장 앞의 원소를 활용하고 큐에서 버리고 싶다면, 반환하는 함수인 `front()`를 사용해서 값을 먼저 복사(또는 읽기)해둔 뒤에 `pop()`을 호출하여 제거해야 구조가 맞물립니다.

## 3. DOD (Data-Oriented Cache Hit) 관점의 분석 🧠
`std::queue`는 사실 자체적인 메모리 구조를 가진 독립된 자료구조가 아니라, 기존 자료구조를 덮어서 형태만 바꿔주는 **컨테이너 어댑터(Container Adapter)**입니다. 기본적으로 **`std::deque`**(덱)라는 자료구조를 기본 뼈대로 사용합니다.

**❓ 그렇다면 왜 코치님은 지금까지 STL을 쓰지 못하게 하고 '정적 배열'로 직접 구현하게 했을까요?**

`std::deque`는 메모리를 일정한 크기의 블록(Chunk) 단위로 쪼개어 관리하며 내부적으로 동적 할당을 수행합니다. 즉, 데이터가 하나의 거대한 메모리 영역에 완전히 연속되어 이어져 있지 않기 때문에, 배열 인덱스로 순차 접근할 때보다 CPU의 **캐시 미스(Cache Miss)**가 미세하게 더 자주 발생합니다.

반면, 우리가 직접 짠 **"거대한 전역 배열 + head/tail 인덱스 포인터"** 방식은:
1. 메모리 동적 할당/해제(Dynamic Allocation/Deallocation)가 런타임에 발생하지 않아 오버헤드가 0입니다. (프로그램 시작 시 한 번 확보)
2. 모든 메모리가 완전히 연속된(Contiguous) 단일 덩어리이므로, CPU가 램에서 메모리를 긁어올 때의 **Cache Locality(캐시 지역성)** 효율이 극대화됩니다.

따라서 극한의 프레임레이트가 필요한 게임 엔진이나, 연산(삽입/삭제)이 200만 번 이상 초당 발생해야 하는 알고리즘 문제에서는 **원시적인 정적 배열+포인터 큐**가 `std::queue`보다 퍼포먼스 면에서 매우 유리합니다. 

## 4. STL Queue 실습 검증 코드
이 코드는 직접 타이핑해보시면서 API의 역할을 눈으로 증명하시기 바랍니다.

```cpp
#include <iostream>
#include <queue> // queue를 사용하기 위한 헤더

using namespace std;

int main() {
    // int형 데이터를 보관하는 큐 생성
    queue<int> q;

    // 데이터 밀어넣기 (push)
    q.push(10);
    q.push(20);
    q.push(30);

    // 큐 상태 확인
    cout << "Size : " << q.size() << "\n";     // 출력: 3
    cout << "Front : " << q.front() << "\n";   // 출력: 10
    cout << "Back : " << q.back() << "\n";     // 출력: 30

    cout << "\n요소 하나씩 꺼내기:\n";
    // 큐가 완전히 텅 상태가 될 때까지 반복
    while (!q.empty()) { 
        cout << q.front() << " "; // 맨 앞 요소 읽기
        q.pop();                  // 읽은 요소 제거하기
    }

    return 0;
}
```

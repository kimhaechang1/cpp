# 🎯 Pointer Reassignment Visualized

## 상황: `head`라는 깃발을 옮겨라!

`head`는 노드가 아니라 **"누가 1등인지 가리키는 손가락(Pointer)"**입니다.

### 1단계: 초기 상태
`head` 손가락이 **[10]**이라는 박스를 가리키고 있습니다.
```
      (head) 
        👇
      [ 10 ] -> [ 20 ] -> nullptr
```

### 2단계: 신입 등장 (`newNode`)
새로운 박스 **[5]**를 만들었습니다. 아직은 외톨이입니다.
```
(newNode)
   👇
 [ 5 ]          (head)
   |              👇
   ?            [ 10 ] -> [ 20 ] -> nullptr
```

### 3단계: 신입이 기존 대장을 잡음 (`newNode->next = head`)
**[5]**의 뒷고리(`next`)를 `head`가 가리키던 **[10]**에 겁니다.
```
(newNode)       (head)
   👇             👇
 [ 5 ] -------> [ 10 ] -> [ 20 ] -> nullptr
```
이제 연결은 됐지만, 아직 `head` 손가락은 여전히 **[10]**을 가리키고 있습니다.
사람들한테 "야, 대장이 누구야?" 물어보면 여전히 "head가 가리키는 10번이요"라고 하겠죠.

### 4단계: 깃발 옮기기 (`head = newNode`)
이제 `head` 손가락을 **[5]**로 옮깁니다.
```
(newNode)       (head)
   👇             👇
 [ 5 ] -------> [ 10 ] -> [ 20 ] -> nullptr
  👆
(이제 얘가 대장!)
```

### 결론
`head`는 상자(Node) 자체가 아닙니다. **"누가 맨 앞인지 가리키는 화살표 변수"**일 뿐입니다.
화살표의 방향을 `[10]`에서 `[5]`로 바꿔주는 행위가 바로 `head = newNode` 입니다.

# C++ 언어 기초 학습 커리큘럼

## 📘 개요
단순한 문법 암기가 아닌, **"게임 개발을 위한 실전 C++"**을 목표로 합니다.
각 단계는 **개념 학습 → 실습 문제 → 미니 프로젝트(스켈레톤 코드 채우기)** 순서로 진행됩니다.

---

## 📚 학습 로드맵 (총 4단계)

### 🌱 BEGINNER (입문)
**목표**: 데이터를 변수에 담고, 조건문과 반복문으로 게임 로직을 제어할 수 있다.

#### MODULE 1: C++ 시작하기
- [ ] **변수와 자료형**: `int`, `float`, `bool`, `sizeof`
- [ ] **사용자 정의 자료형**: `struct` (스탯 구조체 만들기), `enum` (게임 상태 정의)
- [ ] **입출력**: `cin`, `cout`

#### MODULE 2: 제어 흐름
- [ ] **조건문**: HP에 따른 상태 분기 (`if`, `switch`)
- [ ] **반복문**: 턴제 전투의 기본 루프 (`while`, `for`)
- [ ] **흐름 제어**: `break`, `continue`

#### MODULE 3: 함수
- [ ] **함수 기초**: 데미지 계산 함수 분리
- [ ] **매개변수 전달**: `Call by Value` vs `Call by Reference`
- [ ] **재귀 함수**: 팩토리얼(경험치 테이블 등)

---

### 🔧 INTERMEDIATE (중급)
**목표**: 메모리를 직접 조작하고 관리하여, 고정된 크기의 배열을 넘어선다.

#### MODULE 4: 포인터와 참조 (가장 중요 ⭐)
- [ ] **포인터 기초**: 메모리 주소의 이해
- [ ] **포인터와 배열**: 포인터 연산으로 배열 접근하기
- [ ] **함수 포인터**: 스킬 버튼에 함수 연결하기 (콜백)

#### MODULE 5: 동적 할당
- [ ] **메모리 구조**: Stack vs Heap 이해
- [ ] **동적 할당**: `new`, `delete`로 런타임에 배열 크기 정하기
- [ ] **메모리 누수**: 잃어버린 메모리 찾기 (RAII 전단계)

---

### 🏗️ ADVANCED (고급)
**목표**: "설계"를 한다. 데이터와 기능을 묶어 관리하고, 상속을 통해 코드를 재사용한다.

#### MODULE 6: 클래스와 객체
- [ ] **클래스**: `struct`와 `class`의 차이, 접근 지정자 (`private`, `public`)
- [ ] **생성자/소멸자**: 객체의 탄생과 죽음 관리
- [ ] **복사 생성자**: 얕은 복사 vs 깊은 복사

#### MODULE 7: 상속과 다형성
- [ ] **상속**: `Monster`를 상속받는 `Slime`, `Orc` 만들기
- [ ] **가상 함수**: `virtual Attack()`으로 다형성 구현하기
- [ ] **순수 가상 함수**: 인터페이스 설계

---

### ✨ EXPERT (심화)
**목표**: 더 안전하고, 더 빠르고, 더 편리한 "모던 C++" 기법을 익힌다.

#### MODULE 8: 템플릿과 STL
- [ ] **템플릿**: 타입에 상관없는 `Box<T>` 만들기
- [ ] **STL**: `std::vector`, `std::map` 으로 인벤토리/타일맵 구현

#### MODULE 9: 모던 C++
- [ ] **스마트 포인터**: `unique_ptr`, `shared_ptr` (delete 없는 세상)
- [ ] **이동 시맨틱**: 무거운 리소스의 소유권 이전 (`std::move`)

#### MODULE 10: 프로페셔널 도구 (Toolchain)
- [ ] **Build Systems**: CMake를 사용하여 멀티 플랫폼 프로젝트 구성하기.
- [ ] **Package Manager**: Vcpkg나 Conan을 통해 외부 라이브러리(fmt, spdlog 등) 연동하기.
- [ ] **Profiling & Sanitizers**: AddressSanitizer(ASan)로 메모리 오염 자동 탐지하기.
